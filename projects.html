Online POS System

A modern store management solution integrating inventory, sales, and reporting in one intuitive platform.

Project Overview

The Online POS System is a web-based solution designed to optimize inventory management and sales tracking for businesses of all sizes. Built with PHP, MySQL, and Bootstrap, it features an intuitive interface and robust backend capabilities to help businesses manage stock, process sales, and generate informative reports.

Feature Highlights

Main Login Page

A secure login interface for admin and cashier roles. Admins have full access to management features, while cashiers can manage sales transactions and reports.

Main Cashier (Checkout & Payment)

An easy-to-use checkout feature allows cashiers to search for products, add them to the cart, apply discounts, and complete payments (Cash & QRIS).

Cashier Sales Results

Displays sales reports per order with complete details, receipt printing options, and change details for cash transactions.

Main Admin (Dashboard)

The admin dashboard provides key summaries such as income, product count, and total stock. Interactive charts display revenue trends.

Manage User

User management page allowing admins to add, edit, and delete users with various access levels.

Manage Products

Product management module with CRUD features, image upload, barcode scanning, Excel import, pagination, and search.

Inventory Log

Monitor stock movements, including incoming goods, to ensure data accuracy and support traceability.

Payment Logs

Payment transaction records for admins to ensure financial reconciliation, including method, date, and amount details.

Sales Report (PDF)

Provides daily and monthly reports in downloadable PDF format for evaluating sales and product performance.

Export Data (Excel)

Allows admins to export product and sales data to Excel format for in-depth analysis, reporting, and auditing.
