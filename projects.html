Sistem Kasir Online

Solusi lengkap untuk manajemen toko dan pemrosesan penjualan

Gambaran Proyek

Sistem Kasir Online adalah solusi berbasis web yang dirancang untuk mengoptimalkan manajemen inventaris dan pelacakan penjualan bagi bisnis dari berbagai skala. Dibangun menggunakan PHP, MySQL, dan Bootstrap, sistem ini menghadirkan antarmuka yang intuitif dengan kemampuan backend yang kuat untuk membantu bisnis dalam mengelola stok, memproses penjualan, serta menghasilkan laporan yang informatif

Fitur unggulan dari sistem ini meliputi pembaruan stok real-time, pemindaian barcode, akses multi-pengguna dengan izin berbasis peran, serta alat laporan yang dapat disesuaikan. Desain responsif memungkinkan pengguna untuk mengakses dan mengelola inventaris dari berbagai perangkat, menjadikannya solusi ideal bagi bisnis dengan banyak lokasi atau tim yang bekerja jarak jauh.

Project Highlights

Halaman Utama Login Page

Antarmuka login yang aman untuk akses peran admin dan kasir. Admin memiliki akses penuh ke fitur manajemen, sementara kasir dapat mengelola transaksi penjualan dan laporan.

Kasir Utama (Checkout, Diskon, Pembayaran Tunai dan QRIS)

Fitur checkout yang mudah digunakan memungkinkan kasir untuk mencari produk, menambahkannya ke keranjang, menerapkan diskon, dan menyelesaikan pembayaran. Mendukung berbagai metode pembayaran, termasuk tunai dan QRIS, dengan konfirmasi modals untuk setiap transaksi.

Kasir Hasil Penjualan

Menampilkan laporan penjualan berdasarkan ID pesanan dengan rincian lengkap, termasuk jumlah, harga, total setelah diskon, serta metode pembayaran. Kasir dapat mencetak struk dan melihat rincian uang konsumen serta kembalian untuk transaksi tunai.

Admin Utama (Dashboard)

Dashboard admin memberikan ringkasan penting seperti penghasilan hari ini, jumlah produk, dan total stok dalam bentuk kartu. Grafik interaktif menampilkan penghasilan bulanan serta perbandingan mingguan, memudahkan admin untuk memantau kinerja dan tren.

Manage User

Halaman manajemen pengguna yang memungkinkan admin untuk menambah, mengedit, dan menghapus pengguna. Mendukung berbagai level akses untuk mengamankan data sensitif dan pengelolaan tugas yang lebih efisien.

Manage Barang

Modul pengelolaan barang dengan fitur CRUD, upload gambar, dan pemindaian barcode. Fitur tambahan seperti impor data dari file Excel, pagination, dan pencarian memudahkan pengelolaan inventaris. Juga terdapat opsi untuk menghapus semua data produk.

Inventory Log

Memantau pergerakan stok barang, termasuk barang yang masuk. Laporan ini penting untuk memastikan stok selalu diperbarui dan mendukung penelusuran saat ada perubahan jumlah barang.

Payment Logs

Catatan transaksi pembayaran yang dapat dilihat oleh admin untuk memastikan rekonsiliasi keuangan. Laporan ini mencakup rincian metode pembayaran, tanggal, serta jumlah yang diterima.

Laporan Penjualan (PDF)

Menyediakan laporan harian dan bulanan dalam format PDF yang dapat diunduh. Laporan ini membantu manajemen untuk mengevaluasi penjualan dan kinerja produk berdasarkan periode tertentu.

Export Data (Excel)

Memungkinkan admin untuk mengekspor data produk dan penjualan dalam format Excel. Fitur ini mendukung analisis mendalam dan pelaporan secara manual atau otomatis, serta mempermudah pencatatan dan audit.
