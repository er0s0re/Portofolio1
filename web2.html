Learning Journey (5 Parts)

Design & Planning

Building the foundation with a Hub-and-Spoke topology. Choosing the right devices (Router vs Switch) and cable types (Cross vs Straight) for each connection.

IP Address Assignment

Giving a unique identity to each router 'door'. Using the `ip address` command and enabling ports with `no shutdown` to build basic connectivity.

Routing & Network Mapping

Becoming the "GPS" for data packets. Configuring Static and Default Routes (`ip route`) so all networks can communicate without getting lost.

Wireless & DHCP Server

Setting up access for users. Configuring SSID & WPA2-PSK on the Access Point, and setting up the DHCP Server on the router for automatic IP allocation.

Simulation & Admin Control

Testing real scenarios. Simulating users moving buildings and becoming the "Tyrant Admin" by disconnecting connections via `shutdown` and Access Control List (ACL).

Final Result & Technologies

Final Network Topology

Technologies Used

  • Static & Default Routing

    Building a manual network map so all segments are connected.

  • DHCP Server on Router

    Automating IP address allocation for easy client management.

  • WLAN & WPA2-PSK

    Providing secure wireless connectivity for end users.

  • Access Control List (ACL)

    Traffic filtering for advanced access control and security.
