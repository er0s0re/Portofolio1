Case Study
Building an Office Network from Scratch

The journey of designing and implementing a functional, secure, and automated network using Cisco Packet Tracer.

Start Journey View Project

Learning Journey (7 Parts)

1

Basic Connectivity

Understanding the most basic concept: connecting two PCs directly using a Cross-Over cable and performing a `ping` test for the first time.

2

Building a LAN with Switch

Leveling up by using a Switch to build a Local Area Network (LAN) with a Star topology, efficiently connecting many PCs.

3

VLAN Segmentation

Starting to play with network security and neatness. Splitting one physical LAN into two isolated virtual LANs (VLAN 10: IT, VLAN 20: Marketing).

4

Inter-VLAN Routing

Becoming the network "matchmaker". Using Router-on-a-Stick to bridge communication between different VLANs in a controlled manner.

5

Automatic DHCP Server

Becoming an efficient "lazy person". Configuring the Router as a DHCP server to automatically assign IP addresses to all PCs in each VLAN.

6

ACL Security

Becoming a "strict security guard". Applying Access Control List (ACL) to filter traffic, allowing IT to access Marketing, but not vice versa.

7

Internet Connection (NAT)

Opening the gateway to the outside world. Configuring NAT so all PCs with Private IPs can connect to the simulated "internet" network.

Final Result & Technologies

Final Network Topology

Technologies Used

  • Cisco 2911 Router & 2960 Switch

    Core devices for routing, switching, and security.

  • VLAN & Inter-VLAN Routing

    Network segmentation for efficiency and security.

  • DHCP Server on Router

    Automated IP address allocation for easy management.

  • Access Control List (ACL)

    Traffic filtering for access control between VLANs.

  • Network Address Translation (NAT)

    Connecting the local network to the global internet.
