Rikos

Comfortable and Structured, Rikos is Your Boarding Room Booking Solution

Project Overview

Rikos is a web platform designed to simplify the process of booking boarding rooms online. With Rikos, tenants can easily register, log in, and choose rooms according to their needs. Admins have high-level access to manage user data, rooms, and transaction reports. Features such as admin management, user management, clear booking flow, room availability, and booking proof make transactions and room management easier. With Rikos, the process of booking boarding rooms becomes more efficient, effective, and structured.

Rikos also provides features that make it easier for users to book rooms. Users can easily edit their personal profiles, view room availability information, make room bookings, and view transaction history. In addition, Rikos displays information on the availability of male and female rooms, as well as room details including facilities and prices. With Rikos, the process of booking boarding rooms becomes easier, faster, and more structured.

Project Highlights

Main Page

This is the first page displayed when users open the Rikos site. It shows general information about Rikos, available features, and how to use the site.

Profile (Admin)

This page is displayed when the admin logs in. It makes it easy for the admin to manage user data, rooms, and transaction reports. The admin can add, view, edit, and delete user data, as well as view and manage room data.

Payment Validation Form

The Payment Validation Form page is used to validate payments made by users. This page makes it easy for the admin to check payments made by users and validate those payments.

Admin Room Info

The Admin Room Info page is intended for admins who want to view room availability information. This page makes it easy for admins to see room availability, such as the number of available rooms, prices, and facilities available in those rooms.

User Profile Page

The User Profile Page is intended for users who want to edit their personal profile. This page makes it easy for users to edit personal information, such as name, address, phone number, and email.

Empty Room Information Page

The Empty Room Information Page is intended for users who want to view information about available empty rooms. This page makes it easy for users to see the availability of empty rooms, such as the number of available rooms, prices, and facilities in those rooms.

Payment Report Page

The Payment Report Page is intended for admins who want to view payment reports made by users. This page makes it easy for admins to see payment reports made by users, such as the amount paid, payment date, and total paid.
